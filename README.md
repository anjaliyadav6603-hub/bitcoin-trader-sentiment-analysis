# Bitcoin Trader Performance Analysis Based on Market Sentiment

## Overview

This project analyzes over **211,000 historical trades** from Hyperliquid alongside the **Bitcoin Fear & Greed Index** to study how market sentiment impacts trader performance.

The project explores:

- Profitability during Fear vs Greed
- Leverage usage
- Coin-wise performance
- Buy vs Sell analysis
- Trader ranking
- Trading behavior
- Business recommendations

---

## Dataset

### Historical Trader Data

Contains

- Account
- Coin
- Leverage
- Side
- Closed PnL
- Trade Size
- Timestamp

### Fear & Greed Dataset

Contains

- Date
- Fear / Greed Classification
- Index Value
